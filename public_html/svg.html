Drawing Basic Shapes with SVG

Line

To draw a line in SVG we can use the line element. This element is used to draw a single straight line, so it will only consist of two points, start and end. The line start point coordinate is expressed with the first two attributes x1 and y1, while the line’s end point coordinate is expressed with x2 and y2. There are also two other attributes, the stroke and stroke-width which are used to define the border’s color and border’s width.

Polyline

It’s almost similar to the line, but with the polyline element we can draw multiple lines instead of just one. Polyline element has points attributes that store all the coordinates that form the lines.

Rectangle

Drawing a rectangle is also simple with rect element. We only need to specify the width and height. You can also specify start point with x and y.

Circle

We can also draw a circle with the circle element. Radius is defined with r attribute. Attributes cx and cy are defining the circle’s center coordinate.

Ellipse

We can draw an ellipse with ellipse. It works quite similar to the circle, but this time we can control specifically the x line radius and y line radius with rx and ry attribute.

Polygon

With the polygon element, we can draw multiple sides of shapes like a triangle, hexagon and even a rectangle. Polygon element has points attributes that store all the coordinates.

Using Vector Graphic Editor

As you can see, drawing simple objects with SVG in HTML is quite easy. However, when the object gets more complex, that practice is no longer ideal and will give you a headache.
Fortunately, we can use a vector graphic editor like Adobe Illustrator or Inkscape to do the job. If you are working with Inkscape, you can save your vector graphic as plain SVG, and then open it in text code editor. Now, you should find the SVG codes in the file. Copy all the codes and paste them inside your HTML document.

Or, you can also embed the .svg file through one of these elements: embed, iframe and object. The results will eventually be the same.
